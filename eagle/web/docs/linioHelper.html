Linio帐号绑定

Linio平台共在8个拉美国家开通消费者购物站点。其中已开通国际业务（即平台向非拉美本地的商户开放业务）的5个站点是：墨西哥，哥伦比亚，秘鲁，智利，巴拿马。，我以其中的墨西哥站点为例，进行店铺授权。

1.阿根廷 https://sellercenter.linio.com.ar/

2.智利 https://sellercenter.linio.cl/

3.哥伦比亚 https://sellercenter.linio.com.co/

4.厄瓜多尔 https://sellercenter.linio.com.ec/

5.墨西哥 https://sellercenter.linio.com.mx/

6.巴拿马 https://sellercenter.linio.com.pa/

7.秘鲁 https://sellercenter.linio.com.pe/

8.委内瑞拉 https://sellercenter.linio.com.ve/

Step 1 先登录linio后台，进入Settings -Manager Users ，如下图所示

Step 2 添加一个新用户用于对接小老板

注意：是Add User，而不是账号信息后面直接点编辑

Step 3 ：“Role”选择第四项“Seller API Access” ，“Email Address”对应小老板里新建Linio账号的API账号邮箱，“Default Language”选英文

Step 4 点击save后，将出现下图两个用户，复制刚才新增的账号（role为Seller API Access 的账号）的Email和API Key

Step 5 到小老板新建Linio账号

注意：新建账号时，API账号邮箱 , API key 对应linio sellercenter - Manager Users里role 为"Seller API Access的账号的Email Address和 API key，对应的站点也一定要选择。
