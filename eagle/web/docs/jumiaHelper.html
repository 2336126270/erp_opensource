Jumia帐号绑定

Jumia平台共在11个非洲国家开通消费者购物站点。以下将会介绍如何对店铺授权。

1.安哥拉 https://sellercenter.dz.jumia.com/

2.喀麦隆 https://sellercenter.jumia.cm/

3.埃及 https://sellercenter.jumia.com.eg/

4.加纳 https://sellercenter.jumia.com.gh/

5.科特迪瓦 https://sellercenter.jumia.ci/

6.肯尼亚 https://sellercenter.jumia.co.ke/

7.摩洛哥 https://sellercenter.jumia.ma/

8.尼日利亚 https://sellercenter.jumia.com.ng/

9.塞内加尔 https://sellercenter.jumia.sn/

10.坦桑尼亚 https://sellercenter.jumia.co.tz/

11.乌干达 https://sellercenter.jumia.ug/

12.南非 https://sellercenter.jumia.co.za/

Step 1 先登录jumia后台，进入Settings -Manager Users ，如下图所示

Step 2 添加一个新用户用于对接小老板

注意：是Add User，而不是账号信息后面直接点编辑

Step 3 ：“Role”选择第四项“Seller API Access” ，“Email Address”对应小老板里新建Jumia账号的API账号邮箱，“Default Language”选英文

Step 4 点击save后，将出现下图两个用户，复制刚才新增的账号（role为Seller API Access 的账号）的Email和API Key

Step 5 到小老板新建Jumia账号

注意：新建账号时，API账号邮箱 , API key 对应jumia sellercenter - Manager Users里role 为"Seller API Access的账号的Email Address和 API key，对应的站点也一定要选择。
